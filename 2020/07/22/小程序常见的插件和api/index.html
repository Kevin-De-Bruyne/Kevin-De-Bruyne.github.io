小程序常见的插件和api

发表于 2020-07-22

一.enablePullDownRefresh

在小程序里面加上一个下拉刷新的原因是为了优化用户的体验，有时候会遇到这样的一种情况，用户的网络比较慢，那么该出现的内容就出现的很慢，如果有下拉刷新的功能，就会给人一种需要等待一下下一秒就会出现内容的感觉，而不是空白页或者这个小程序有bug之类的疑问。

二.wx.stopPullDownRefresh(Object object)

停止当前页面下拉刷新。

三. wx.showLoading () wx.hideLoading()

​ 弹出加载框

四.wx.showToast(Object object)

弹出遮罩提示

五.wx.previewImage(Object object)

在新页面中全屏预览图片。预览的过程中用户可以进行保存图片、发送给朋友等操作。

六.wx.setStorageSync(string key, any data)

需要存储的内容。只支持原生类型、Date、及能够通过JSON.stringify序列化的对象。

七.wx.getStorageSync(string key)

string key

本地缓存中指定的 key

八.wx.chooseAddress(Object object)

获取用户收货地址。调起用户编辑收货地址原生界面，并在编辑完成后返回用户选择的地址。

九.wx.showModal(Object object)

弹出对话框

十.wx.chooseImage(Object object)

从本地相册选择图片或使用相机拍照。

十一. wx.uploadFile(Object object)

将本地资源上传到服务器。客户端发起一个 HTTPS POST 请求，其中 content-type 为 multipart/form-data。使用前请注意阅读相关说明。
